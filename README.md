### <영화종합 웹사이트>

### 이 프로젝트는 팀프로젝트이후 개인적으로 기능추가및 개선(ux/ui) 한 프로젝트 입니다.
#### 아직배포는 못했으며, 웹호스팅을 먼저해볼생각입니다.(차후에)
### 몇몇 코드는 정리가 안되어있으며, 현재 사용하지 않는 코드도 포함되어 있습니다.(기능이 안되진 않음).(css분리가 안된코드도 몇 존재)

#### 예시: `check_movie_update_needed()` (inc/db.php)
- 이 함수는 정상적으로 작동하지만, API 호출이 많아 응답 속도가 느려지는 문제가 있었습니다.
- 현재는 관리자 수동 업데이트 방식을 사용 중입니다.
- 코드를 삭제하지 않은 이유: 기능 자체는 동작하며, 다른 구현 방식의 예시로 남겨두었습니다.


### 전체적으로 기존의방식 코드를 지우지않은 파일이 몇개존재함. 이코들들은 기능을꺼둔상태(기능은됨) 단지 현재 방식이더 효울적이라서 그래서 주석처리된것도있고 그대로 쓴것도있음(어차피막음) 이런 식으로 접근가능하다라는걸 남기기위해서 냅둠 추후에 주석처리를 해야하던지 따로 파일을분리해서 저장하는방법을 할계획.
## ***따라서 코드확인하실떄 현재쓰는방식을 잘보셔야 합니다.***
