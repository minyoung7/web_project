### 코드 구조 안내
일부 **이전 구현 코드**가 포함되어 있습니다. 현재 사용하지 않지만, 다른 구현 방식의 참고용으로 보존했습니다.

#### 예시: `check_movie_update_needed()` (inc/db.php)
- 이 함수는 정상적으로 작동하지만, API 호출이 많아 응답 속도가 느려지는 문제가 있었습니다.
- 현재는 관리자 수동 업데이트 방식을 사용 중입니다.
- 코드를 삭제하지 않은 이유: 기능 자체는 동작하며, 다른 구현 방식의 예시로 남겨두었습니다.
- 개선 과정은 포트폴리오 문서에서 확인할 수 있습니다.

### 추후 계획
전체적으로 이전 구현 코드를 지우지않은 파일이 몇개존재함. 이 코드들은 기능을꺼둔상태(기능은됨) 단지 현재 방식이더 효율적이라서 그래서 주석처리된것도있고 그대로 쓴것도있음(어차피막음) 이런 식으로 접근가능하다라는걸 남기기위해서 냅둠 추후에 주석처리를 해야하던지 따로 파일을분리해서 저장하는방법을 할계획.
