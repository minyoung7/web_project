### <영화종합 웹사이트>

### 이 프로젝트는 팀프로젝트 이후 개인적으로 기능추가 및 개선(ux/ui) 한 프로젝트 입니다.
#### 아직배포는 못했으며, 웹호스팅을 먼저해볼생각입니다.(차후에)

### 코드 구조 안내
일부 **이전 구현 코드**가 포함되어 있습니다. 현재 사용하지 않지만, 다른 구현 방식의 참고용으로 보존했습니다.

#### 예시: `check_movie_update_needed()` (inc/db.php)
- 이 함수는 정상적으로 작동하지만, API 호출이 많아 응답 속도가 느려지는 문제가 있었습니다.
- 현재는 관리자 수동 업데이트 방식을 사용 중입니다.
- 코드를 삭제하지 않은 이유: 기능 자체는 동작하며, 다른 구현 방식의 예시로 남겨두었습니다.
- 개선 과정은 포트폴리오 문서에서 확인할 수 있습니다.

### 추후 계획
일부 파일에 이전 구현 코드가 남아있습니다.  
해당 코드들은 기능이 비활성화된 상태이며(기능 자체는 정상 작동), 현재 방식이 더 효율적이라 대체되었습니다.  
다른 구현 방식의 참고용으로 보존 중이며, 추후 주석 정리 또는 별도 파일로 분리할 예정입니다.

### ***현재 사용 중인 코드는 포트폴리오 문서 내용과 일치합니다.***
※ 포트폴리오 문서의 코드는 설명을 위해 핵심 로직만 발췌한 것이며, 실제 코드는 수정 없이 그대로 업로드되어 있습니다.

DB 테이블 구조는 sql/moviedb.sql 파일에 정리되어 있습니다.

※ 이미지 파일(광고, 영화 포스터 등)은 용량 및 저작권 문제로 업로드하지 않았습니다.
